scrapy_starter
===========================

Requirements
---------
* pip
* scrapy
* sqlalchemy
* sqlite

Install
---------

~~~ sh
$ pip install scrapy
$ pip install sqlalchemy
~~~

Crawling
---------

~~~  sh
$ scrapy crawl juzi
~~~
